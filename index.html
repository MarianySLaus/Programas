Linguagens de Programação

Esse termo é usado para se referir a um conjunto de instruções e regras que são usadas para fazer um programa (software). Vamos ver algumas das diferenças entre: Linguagens de Baixo nível e Alto nível.


Linguagem de Baixo-nível

Pode-se dizer que são as linguagens com maior dependencia do hardware, não se ajustando para qualquer tipo de plataforma. Isso faz com que seu código fique menos portável e dependa de mais ajustes. Um exemplo é a linguagem Assembly.


Linguagem de Alto-nível

As linguagens de alto-nível são mais portáveis, o que permite que o código seja executado em diversos sistemas sem precisar de muitas alterações. Alguns exemplos são a linguagem C++ e Java.
